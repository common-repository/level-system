== Description ==

The one and only Level System for WordPress. Just add the widget to your favorite area and enjoy the fully customizable and responsive level system. Pick your own colors and width for the progress bar. Automatically give users on your website level and experience points as they contribute to your website. Possibilities to test all XP points & levels directly in your browser.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/level-system` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->WP Level System screen to configure the plugin


== Frequently Asked Questions ==

= Can i customize the plugin? =

After activating the level system plugin you can go to settings->WP Level System screen to change the width of the progress bar, change color of the progress bar, change color of the text within the progress bar and change color of the text within the level system widget.

= Can i test all the different levels, experience points and width of the progress bar? =

Yes you do this by simply adding e.g.: 

?xp=replaceThisTextWithDifferentNumbersYouWantToTest

To the end of the URL where the level system is currently showed.
